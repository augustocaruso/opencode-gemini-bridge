# Security checklist

- [ ] No API keys in repo.
- [ ] No auth.json in repo.
- [ ] No OAuth refresh tokens in logs.
- [ ] MCP env values masked in doctor.
- [ ] Hooks disabled by default.
- [ ] Extension scripts require trust.
- [ ] `yolo` is not global default.
- [ ] `external_directory` is ask/deny by default.
- [ ] Generated files are clearly marked.
- [ ] Dry-run available for sync/install.
- [ ] Windows install does not overwrite user files without backup.
